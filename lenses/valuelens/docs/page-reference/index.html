    Page reference

    ValueLens ships as a single Power BI report. Its fullest build carries a 15-page portfolio — adoption, value, maturity, governance, feedback, billing, and reference appendices. Page availability varies by deployment path: the SharePoint and base Fabric builds carry a subset, and the Copilot Studio agent pages appear only on the Fabric Extended build. Where a page name differs by path, each source’s own label is preserved below.

    The full page portfolio

    The 15 pages of the fullest (Fabric Extended) build:

    User Activation

    Activation across teams: licensed vs unlicensed, active vs inactive.

    Adoption & Reach

    User counts, coverage percentage, and licensed vs unlicensed reach.

    Activity & Value

    Copilot and agent usage, tasks, hours saved, and dollar-equivalent assisted value with the business case.

    Usage Maturity

    The Asking to Finding to Consuming to Producing to Delegating maturity progression.

    Leaderboards

    Top users, agents, and functions.

    Agent Governance

    Deployment patterns, creator insights, and sensitivity exposure.

    User Feedback

    Thumbs up/down sentiment and feedback themes.

    License Readiness

    Ranks unlicensed and low-adoption users by upgrade-priority score.

    Heatmap Trend

    The activity heatmap across the reporting period.

    Credits Consumed

    Copilot Studio agent credit consumption and billing breakdown. (Fabric Extended — Copilot Studio pages.)

    Agent Evaluation

    Agent resolution, abandonment, escalation, and response time. (Fabric Extended — Copilot Studio pages.)

    Topic Analysis

    Most-asked topics, resolution, and abandonment by agent. (Fabric Extended — Copilot Studio pages.)

    Key Concepts

    Methodology and key-concept explainers.

    Glossary

    Metric definitions and research sources.

    Signal Table

    Trace raw signals through to value (the audit trail).

    Availability by deployment path

    Page availability and some page names differ by path. These are each source’s exact inventories — do not assume the SharePoint or base Fabric builds include every extended page.

    SharePoint build (13 pages)

    Activation, Readiness, Adoption, Consumption, Activity, Value, Maturity, Agent Health, Heatmap, Leaderboard, Appendix: Key Concepts, Appendix: Glossary, Appendix: Signal Table.

    Base Fabric build (14 pages)

    Activation, Readiness, Adoption, Consumption, Activity, Value, Maturity, Agent Health, Feedback, Heatmap, Leaderboard, Appendix: Key Concepts, Appendix: Signal - Impact Table, Appendix: Glossary.

    Fabric Extended build (fullest portfolio)

    The full 15-page portfolio above, adding the Copilot Studio pages — Credits Consumed, Agent Evaluation, and Topic Analysis — which are not present on the SharePoint or base Fabric builds.

    The base builds render Activity and Value as separate pages; the fullest portfolio groups them as Activity & Value. Names such as Signal Table (SharePoint) versus Signal - Impact Table (base Fabric) are each path’s exact label.
