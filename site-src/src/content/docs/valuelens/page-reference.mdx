ValueLens ships as a single Power BI report. Its **fullest build carries a 15-page portfolio** —
adoption, value, maturity, governance, feedback, billing, and reference appendices. **Page
availability varies by deployment path:** the SharePoint and base Fabric builds carry a subset, and
the Copilot Studio agent pages appear only on the Fabric Extended build. Where a page name differs by
path, each source's own label is preserved below.

## Availability by deployment path

Page availability and some page names differ by path. These are each source's exact inventories — do
not assume the SharePoint or base Fabric builds include every extended page.

### SharePoint build (13 pages)

Activation, Readiness, Adoption, Consumption, Activity, Value, Maturity, Agent Health, Heatmap,
Leaderboard, Appendix: Key Concepts, Appendix: Glossary, Appendix: Signal Table.

### Base Fabric build (14 pages)

Activation, Readiness, Adoption, Consumption, Activity, Value, Maturity, Agent Health, Feedback,
Heatmap, Leaderboard, Appendix: Key Concepts, Appendix: Signal - Impact Table, Appendix: Glossary.

### Fabric Extended build (fullest portfolio)

The full 15-page portfolio above, adding the Copilot Studio pages — Credits Consumed, Agent
Evaluation, and Topic Analysis — which are not present on the SharePoint or base Fabric builds.

> The base builds render Activity and Value as separate pages; the fullest portfolio groups them as
> **Activity & Value**. Names such as **Signal Table** (SharePoint) versus **Signal - Impact Table**
> (base Fabric) are each path's exact label.
